Welcome to the Trash Pickup Demonstration Walkthrough



These examples are intended to show how evolutionary algorithms work in a non-technical way

This walkthrough will guide you through how to use this example

Learn how to evolve new generations, run simulations, and refresh the trash grid



You will see with this walkthrough:
  • "OK" buttons
  • information popups
  • buttons with animation
  • automatic screen scrolling
  • a complete button

The demo without the walkthrough functions exactly the same, but is bare. You may start it again at any time by clicking the "W" button located in the upper-right area of the page

Reading the little bit of information provided all throughout this site will allow for the best experience for later
